## 1. ToC — 個人ユーザー（市民・ファミリー）

### 1A. 健康・ウェルビーイング
| 訴求 | メッセージ |
|---|---|
| **歩く動機づけ** | 「散歩が"調査"に変わる」— 目的ある歩行で継続率UP |
| **メンタルヘルス** | 自然接触 × 歩行 = ストレス低減（森林浴のエビデンス） |
| **家族の時間** | 子どもと一緒に「発見」を共有する知的レジャー |
| **達成感・成長実感** | バッジ、ストリーク、自己ベスト → ドーパミン設計 |

### 1B. 自然保全への貢献実感
| 訴求 | メッセージ |
|---|---|
| **市民科学** | 「あなたの記録が科学データになる」 |
| **レッドリスト連携** | 絶滅危惧種を発見 → その意味がわかる感動 |
| **リアルタイム可視化** | 地域の生物相がマップ上に蓄積 → 自分の貢献が見える |
| **子どもの科学教育** | 検索キーで「なぜそう分類されるか」を学べる |

## 4. ToA — アカデミア・研究機関

### 4A. 市民科学データ基盤
| 訴求 | メッセージ |
|---|---|
| **広域モニタリング** | 専門調査では届かない「市民の目」で面的データ収集 |
| **長期データ蓄積** | 季節・年単位の定点観測データ → 気候変動影響の追跡 |
| **GBIF連携** | 国際標準フォーマットで共有 → グローバル研究に貢献 |
| **引用元主義** | 学術文献に忠実な分類体系 → 研究者の信頼を獲得 |
| **データライセンス** | 匿名化済み観察データを研究利用に提供 |

### 4B. 教育・アウトリーチ
| 訴求 | メッセージ |
|---|---|
| **フィールドワーク支援** | 大学の野外実習・ゼミ活動のツールとして |
| **科学コミュニケーション** | 「分類とは何か」を市民に伝えるプラットフォーム |
| **共同研究** | 特定地域・分類群の集中調査プロジェクト |
| **学生の研究素材** | 卒論・修論のデータソースとして活用 |

## 5. 財源レイヤー（横断）

### 5A. 補助金・助成金（活用実績 or 対象候補）

| 分類 | 補助金名（例） | 対応セクション |

| **環境省** | 生物多様性保全推進交付金 | 3A |
| **環境省** | 自然共生サイト認定支援 | 2A, 3A |
| **総務省** | 関係人口創出・拡大事業 | 3B |
| **経産省** | 地域DX推進事業 | 3A, 3B |
| **厚労省** | 健康増進事業（自治体経由） | 1A, 3C |
