# Docs Archive

このディレクトリは historical docs の保管場所。通常の ikimon.life 開発では `docs/`, `platform_v2/`, `ops/` を優先し、ここを探索入口にしない。

`docs/archive/` は root `.ignore` で通常の `rg` 対象から外している。過去経緯や rollback 証跡が必要なときだけ `rg -uuu <query> docs/archive` で検索する。
