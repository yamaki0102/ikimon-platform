# ikimon.life B2B戦略 v2：環境コストをデータ資産に変える

> **戦略の核**: 「環境保護はコストではなく投資である」と企業に気づかせる。
> ikimonは観察データを**使える形**で出力するシステム。人的サポートはゼロ。

---

## 1. 戦略の転換：なぜv2が必要か

### v1の問題
v1は「TNFD対応ツール」として設計していた。これは**コンプライアンス売り**（義務だから買って）。
- 顧客の感情：「また金がかかる…」
- 意思決定者：CSR部門（予算小さい）
- 対象市場：TNFD開示義務のある大企業のみ
- 価格の正当化：「他社より安い」（価格競争に巻き込まれる）

### v2の核心
「**環境対応コストを"使えるデータ資産"に変換するプラットフォーム**」。

| 比較 | v1（コンプライアンス売り） | v2（データ資産化売り） |
|:---|:---|:---|
| 顧客の反応 | 「義務だから仕方ない」 | 「え、これ色々使えるの？」 |
| 意思決定者 | CSR部門 | 経営企画・総務・CFO |
| 対象市場 | TNFD義務企業（大企業） | **環境コストを持つ全企業**（中小含む） |
| 単価根拠 | 「他社より安い」 | 「環境投資が資産化される価値」 |
| LTV | レポート出力で終わり | 毎月の観察→毎月のデータ蓄積（永続） |

## 2. コアバリュー：観察会 → データ → レポート

### 企業がやること（たった3ステップ）

```
Step 1: サイトを登録する（工場敷地、社有林、店舗周辺 etc.）
Step 2: 月1回、社員で観察会を開催する
       → ikimonのイベントテンプレートを使えば誰でも運営できる
Step 3: ikimonで記録する（アプリで写真撮って投稿するだけ）
```

### ikimonが自動でやること（全てシステム完結）

```
✅ BISスコア（生物多様性指標）を毎月自動更新
✅ 時系列グラフ（スコア推移、種数推移、季節変動）
✅ レッドリスト自動照合（環境省 + 都道府県）
✅ レポートをワンクリックでPDF出力
✅ 複数フォーマットで出力 ← ★ ここがキモ
```

### レポートテンプレの種類

| テンプレート名 | 主要用途 | 含まれるデータ |
|:---|:---|:---|
| **TNFD LEAP レポート** | TNFD開示義務対応 | LEAP4フェーズ分析、依存度・影響度評価 |
| **サステナビリティレポート** | CSR/統合報告書の根拠資料 | BISスコア推移、種数、SDGsマッピング |
| **活動報告書（汎用）** | 社内報告・外部提出 | 観察実績、参加者数、写真、成果サマリー |
| **エグゼクティブサマリー** | 経営会議・稟議 | 1ページ要約、投資効果、前年比較 |
| **写真付きダイジェスト** | 社内広報・SNS | ベストショット、季節ハイライト |
| **データエクスポート** | 自社分析・他システム連携 | CSV/JSON形式の生データ |

> **設計原則**: 「活動報告書（汎用）」は助成金申請・自治体報告にもそのまま使えるフォーマットだが、ikimonはそれを明示しない。**汎用テンプレート**として提供するだけ。使い方は顧客が自由に決める。

## 3. ポジショニング：誰もいない場所

### 競合マップ

| プレイヤー | 何を売っているか | できないこと |
|:---|:---|:---|
| **NatureMetrics** | eDNA解析（ラボ送付→レポート） | 現場のリアルタイムデータ、社員参加 |
| **IBAT** | 静的な生物多様性スクリーニング | 経年変化の追跡、活動の可視化 |
| **ASUENE** | ESG一括管理（排出量中心） | 現場の生物多様性データ |
| **iNaturalist** | コミュニティ観察プラットフォーム | 企業向けレポート、ダッシュボード |

### ikimonのポジション

> **「現場の観察活動を、企業が使える複数フォーマットのレポートに自動変換する唯一のプラットフォーム」**

競合との決定的な違い：
- NatureMetricsは**ラボに送る**（コスト高い、参加感ゼロ）
- ikimonは**社員が自分で観察する**（コスト低い、チームビルディング、福利厚生）

## 4. 価格戦略

### 設計原則
- 機能制限で課金しない（Community Planで全機能使える）
- Business Planは**レポートの種類とサイト数**で課金
- 人的サポートはゼロ。全てセルフサービス

### プラン設計（2プラン・年間契約のみ）

| | **Community** | **Business** |
|:---|:---|:---|
| **年額** | ¥0 | ¥498,000 |
| **サイト数** | 1 | 無制限 |
| **レポート** | BISスコアのみ | 全テンプレート + PDF出力 |
| **イベント管理** | 基本のみ | 参加者集計 + サマリー自動生成 |
| **データエクスポート** | ❌ | CSV / JSON |

> **年間契約のみ**。月額プランは提供しない。
> 理由：生物多様性データは年間を通じた蓄積で初めて価値が出る。1ヶ月では季節変動すら捉えられない。
>
> **Enterpriseプランは作らない**。需要が出たら後から追加する。今は2プランでシンプルに。

### 価格の正当化ロジック

**Business Plan ¥498,000/年**:
- 企業の環境対応費: 年間300万円以上が一般的
- ikimonのコスト: 年間約50万円（環境予算の17%以下）
- 得られるもの:
  - 毎月のBISスコアレポート（外注すれば1回30万円相当）
  - TNFD対応レポート（コンサルに頼めば100万円以上）
  - CSR報告書の根拠データ（毎年の制作費削減）
  - 社員観察会のテンプレート（チームビルディング費用として計上可能）
  - 活動報告書（**各種外部提出にそのまま使える**）

> 年50万円で、**環境予算全体のROIが劇的に上がる**。
> 判断は二択：**無料で1サイト試すか、全機能使うか**。

## 5. ファネル設計：人を介さないコンバージョン

### サイト構造

```
/for-business             ← LP: 環境コストを資産に変える
  /for-business/demo      ← インタラクティブデモ（既存データで動く）
  /for-business/reports   ← レポートサンプル一覧（全テンプレートのPDF）
  /for-business/pricing   ← 料金プラン
  /for-business/signup    ← セルフサービス申込（決済まで完結）
  /for-business/guide     ← 導入ガイド（観察会の始め方）
```

### コンバージョンフロー（人の介在ゼロ）

```
認知 → LP → デモ体験 → レポートサンプルDL → 料金確認 → 申込 → 即利用開始
                                                              ↑
                                              ここで人間は一切関与しない
```

### 各ページの役割

#### `/for-business` — ランディングページ
**ヘッドライン**: 「観察するだけで、環境投資が"見える"ようになる」

コンテンツ構成:
1. **課題提起**: 「環境対応に年間○○万円。でもその成果、可視化できていますか？」
2. **解決策**: 「月1回の観察会 + ikimon = 6種類のレポートが自動で出る」
3. **3ステップ図解**: 登録 → 観察 → レポート
4. **レポートサンプルのプレビュー**: 各テンプレートのPDFサムネイル
5. **活用シーン一覧**:
   - TNFD開示の根拠資料
   - CSRレポートへの引用
   - 社内の環境KPIダッシュボード
   - 福利厚生としての観察会
   - **各種助成制度への活動報告** ← さりげなく
6. **CTA**: 「無料デモを体験する」「レポートサンプルをダウンロード」

#### `/for-business/demo` — インタラクティブデモ
- 愛管株式会社の実データ（または架空企業データ）でBISスコア・レポートを実際に操作
- ダウンロードはできない（CTA → 申込へ）

#### `/for-business/reports` — レポートサンプル
- 6種類のテンプレートのPDFサンプルを**ダウンロード可能**
- 「このレポートが毎月自動で出ます」のコピー
- リード獲得: メールアドレスを入力してダウンロード

#### `/for-business/guide` — 導入ガイド
- **「観察会の始め方」完全マニュアル**
- 社員向けの案内テンプレート（Wordダウンロード）
- 観察会当日の進行表
- 初心者でもできるikimonの使い方
- → 「サポートなしで観察会が回せる」ことを証明するページ

## 7. 「見せない」補助金戦略

### 直接は言わないが、システムが対応していること

| ikimonの機能 | 助成金側が求めるもの | 接続方法 |
|:---|:---|:---|
| 活動報告書テンプレート | 助成金の成果報告書 | 汎用フォーマットで出力するだけ |
| 参加者数の自動集計 | 参加者延べ人数の報告 | イベント機能の標準出力 |
| 写真付きダイジェスト | 活動写真の添付 | レポートテンプレの1つとして |
| BISスコア推移 | 成果指標の定量証拠 | グラフ付きPDFで出力 |
| レッドリスト照合 | 保全対象種の確認 | レポートに自動記載 |
| CSV/JSONエクスポート | 独自フォーマットへの転記 | データエクスポート機能 |

### ページ上での見せ方

`/for-business` のLPで「活用シーン」セクションに：

```
📊 あなたのレポートが活躍する場面
───────────────────────────────
✅ TNFD開示フレームワークへの入力データ
✅ CSR/サステナビリティレポートの根拠資料
✅ 環境関連の助成制度への活動実績報告
✅ 自治体への環境保全活動報告
✅ 社内のチームビルディング成果レポート
```

## 9. リスクと対策

| リスク | 深刻度 | 対策 |
|:---|:---:|:---|
| BISスコアの信頼性にクレーム | 高 | 算出ロジックの公開＋学術論文の引用 |
| 観察会を継続できない企業 | 中 | 四半期に1回でも価値が出る設計 |
| 大企業がセルフサービスを嫌がる | 中 | 代理店チャネルで間接的にカバー |
| 無料プランだけ使われて課金されない | 低 | Community Planは1サイト制限で十分 |
