# ikimon.life × 地方創生2.0 統合ロードマップ

## Context

地方創生2.0（2025年6月閣議決定）と地方創生総合戦略（2025年12月閣議決定）により、日本の地域政策は「新結合」「AI・デジタル活用」「関係人口」を柱にした10年計画に入った。同時に30by30（自然共生サイト485か所認定済）、地域生物多様性増進法（2025年4月施行）、TNFD開示の制度化が進み、**生物多様性データが「地方の価値」を証明する公式ツール**になりつつある。

ikimon.life は既に BIS/TNFD/30by30対応・100年アーカイブ哲学・消滅可能性自治体無償提供を持つが、**政策フレームワークとの接続が弱い**。このロードマップは既存フェーズ（15B〜18）に地方創生要素を統合し、Phase 19-20 として新機能を追加する。

---

## Phase 別統合計画

### Phase 15B（現在進行中）— Gamification に地方創生フックを追加

**追加要素（小規模・既存作業に統合可能）:**

1. **「地域貢献スコア」バッジ追加** — Gamification.php に以下を追加
   - 「○○市の生物多様性に貢献」バッジ（その自治体エリアで N 種以上記録）
   - 「関係人口バッジ」— 居住地以外の地域で記録した場合に付与
   - ファイル: `upload_package/libs/Gamification.php`

2. **プロフィールに「ふるさと」表示強化**
   - 既に `home_region` が実装済み → これを活用して「あなたのふるさとの生物多様性」セクションを profile.php に追加
   - 「ふるさとに○種の記録があります。あと△種で全国平均を超えます」
   - ファイル: `upload_package/public_html/profile.php`

### Phase 16 — 100年アーカイブ基盤 + **自治体連携の土台**

**既存計画に追加:**

3. **自治体ポートフォリオダッシュボード（MVP）**
   - 新ページ: `upload_package/public_html/municipal_dashboard.php`
   - 表示内容:
     - 市区町村全体の BIS スコア（全登録サイトの加重平均）
     - 30by30 進捗率（認定面積 / 市区町村面積）
     - 記録者数・記録数の推移グラフ（= 関係人口の proxy）
     - 「このデータは○人の市民が記録しました」の参加者可視化
   - データソース: 既存の DataStore + SiteManager
   - 参考: 既存 `site_dashboard.php` の構造をベースに拡張

4. **regional_ecology.json の活用**
   - 既に全都道府県データが存在 → municipal_dashboard で「この地域の特徴的な生態系」を自動表示
   - ファイル: `upload_package/data/regional_ecology.json`

5. **地域生物多様性増進法 対応**
   - 「連携増進活動実施計画」のテンプレート出力機能
   - 既存 ReportEngine.php を拡張して、環境省フォーマットに変換
   - ファイル: `upload_package/libs/ReportEngine.php`

### Phase 17（AI会話型フィールドガイド — 実装済み）+ **地方創生メッセージング**

6. **AI ガイド「愛」に地域文脈を追加**
   - 「この地域は消滅可能性自治体に指定されています。あなたの記録が地域の価値を証明します」
   - 「浜松市のBISスコアは県内3位。あと○種記録すれば2位に」
   - 既存の OmoikaneInferenceEnhancer に地域コンテキスト注入
   - ファイル: `upload_package/libs/OmoikaneInferenceEnhancer.php`

---

### Phase 18 — 生態系インテリジェンス + **RESAS連携**

**既存 Sprint A-E に以下を統合:**

7. **RESAS API 連携（Sprint A に追加）**
   - 新ファイル: `upload_package/libs/RESASConnector.php`
   - 取得データ:
     - 人口構成（年齢・性別）→ 消滅可能性判定の自動更新
     - 観光入込客数 → BIS向上との相関分析
     - 産業構造 → 自然資本依存度の推定
   - RESAS API キー: 無料（自治体・教育機関向け）
   - 表示先: municipal_dashboard.php に「経済インパクト」タブ追加

8. **「関係人口」定量化エンジン**
   - 定義: 居住地外から記録を投稿した人 = 関係人口
   - 指標:
     - 関係人口数（ユニークユーザー数）
     - 関係人口密度（記録数 / 面積）
     - リピート率（2回以上訪問した関係人口の割合）
   - 地方創生の「関係人口」KPI（政府は数値目標を避けた）に対して、ikimon が初の定量データを提供
   - ファイル: `upload_package/libs/RelationPopulationAnalyzer.php`

### Phase 19（新規）— エコツーリズム連携プラットフォーム

9. **季節×地点の生物出現カレンダー**
   - 新ページ: `upload_package/public_html/eco_calendar.php`
   - 蓄積データから「5月の○○市 → ホタル観察の最適時期」を自動生成
   - 多言語対応（日/英）で訪日外国人にもリーチ
   - 地方部訪日外国人宿泊 1.3億人泊（2030年目標）の受け皿

10. **BioBlitz ツーリズム（体験商品化）**
    - 既存 BioBlitz 機能を拡張
    - 「観光 BioBlitz」モード: ガイド付き生物調査体験
    - DMO/観光協会が「体験コンテンツ」としてikimonイベントを掲載可能
    - 予約・問い合わせの外部リンク連携（じゃらん・アソビュー等）

11. **地域ストーリー自動生成**
    - Phase 18 の環境文脈エンジンを活用
    - 「この川沿いには32種の鳥類が記録されています。特に注目はカワセミ。清流のハンターとして知られ...」
    - 多言語ガイドコンテンツとして、観光パンフレット・Web・音声に展開
    - ファイル: `upload_package/libs/StoryGenerator.php`

---

### Phase 20（新規）— 交付金申請支援 + 政策エビデンス

12. **交付金申請エビデンスパッケージ自動生成**
    - 新ページ: `upload_package/public_html/grant_evidence.php`
    - 対象交付金:
      - 地域未来交付金（旧・地方創生交付金）
      - 自然共生サイト認定申請
      - 環境省 生物多様性増進活動認定
    - 出力内容:
      - 地域の生物多様性現況レポート（BIS、種数、希少種）
      - 市民参加の実績データ（記録者数、記録頻度、関係人口）
      - TNFD LEAP 準拠の環境評価
      - 経済インパクト推定（RESAS連携データ）
    - フォーマット: PDF + 政府指定Excelテンプレート

13. **消滅可能性自治体向け「生態系再発見プログラム」**
    - 既存の無償提供モデルを具体化
    - 段階:
      1. 初期セットアップ（サイト登録、初回BioBlitz開催支援）
      2. 3ヶ月データ蓄積 → 初回レポート自動生成
      3. 6ヶ月 → 交付金申請パッケージ生成
      4. 12ヶ月 → 「地域の生態系ポートフォリオ」完成
    - パイロット: 2-3自治体から開始

## 地方創生2.0の5本柱との対応

| 政策の柱 | ikimon.life の対応 |
|---|---|
| 安心して働き暮らせる環境 | ウェルビーイング入口（散歩の豊かさ、自然処方箋） |
| 稼ぐ力・イノベーション | TNFD/BIS → 企業の自然資本開示、エコツーリズム商品化 |
| 人の地方分散・関係人口 | 記録者＝関係人口の定量化、ふるさと住民登録連携 |
| AI・デジタル活用 | BioScan/Gemini/Phase 18 生態系インテリジェンス |
| 広域リージョン連携 | 自治体統合ポートフォリオ、都道府県横断分析 |

## 検証方法

1. **Phase 15B**: 地域貢献バッジが正しく付与されるか手動テスト + `composer test`
2. **Phase 16**: municipal_dashboard.php のレンダリング確認（ローカルサーバー）
3. **Phase 18**: RESAS API のモック → 実API接続テスト
4. **Phase 19-20**: エコカレンダーの季節データ正確性、PDF出力の環境省フォーマット準拠

## 戦略的インパクト

このロードマップが完成すると、ikimon.life は:

- **日本初の「生物多様性 × 地方創生」統合プラットフォーム** になる
- 政府の地方創生交付金申請で「デジタル実装事例」として引用可能
- RESAS や RAIDA と並ぶ「地域分析ツール」として政策議論に参加可能
- 「関係人口」の初の定量的データソースとして、政府のKPI策定に貢献
- 消滅可能性744自治体の「生態系による地域価値再発見」のインフラになる
