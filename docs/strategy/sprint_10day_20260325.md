## リスク

| リスク | 対策 |
|--------|------|
| BirdNET API のレスポンスが遅い（>3秒） | タイムアウト設定 + 「分析中...」表示 |
| Gemini API の quota 超え | 省エネモードでキャプチャ間隔を広げる |
| モバイルブラウザでカメラ+音声同時取得が不安定 | getUserMedia を分離（カメラ→音声の順） |
| Fog of War + LiveScanner の同時動作でメモリ不足 | プロファイリング（Day 5 で実測） |
| 本番サーバーの PHP バージョン差異 | deploy 前に php -l 全ファイル |
