# 愛管株式会社 (I-KAN) 自然共生サイト 概要データ

## 基本情報
*   **サイト名 (内部ID):** I-KAN 本社 (`ikan_hq`)
*   **所有・管理者:** 愛管株式会社
*   **所在地:** 静岡県（都田地域）
*   **面積:** 約 1.3ha
*   **実施区域の状況:** 畑・果樹園・牧草地、創出緑地

## サイトの概要と取り組み
約1.3haのエリアで、以下のような多様な事業と環境が共存しています。
*   管工事業
*   レストラン (LENRI)
*   カフェ
*   農産物加工工房
*   農園（有機農業）
*   認可保育園

**主な役割と環境への配慮:**
1.  **生態系に配慮した農園運営:** 中心となる農園では有機農業を行い、生態系に配慮した環境に優しい農業を実践。外来種の駆除等も実施。
2.  **食農保育と環境教育:** サイト内の保育園やレストランで働く人々が連携し、食農保育を通じて生態系の大切さを学ぶ取り組みを実施。園児と一緒に生物の観察を行うなど、豊かな心を育む場として機能。
3.  **コミュニティ形成:** マルシェ等の開催を通じて、地域住民の憩いの場・コミュニティ形成の場として機能。
4.  **緑化と脱炭素:** サイト内の緑地化の維持と脱炭素化への取り組み。

## 保全・増進活動の目標
自社内の生物多様性保全推進に留まらず、サイトを起点に **都田地域を含めた他企業の生物多様性への取り組み機運醸成** を目標としています。
具体的には：
*   サイトにおける生物多様性保護の高度化
*   企業や保護者に向けた視察等の積極的な誘致
これらを通じて、生物多様性の輪を広げていくことを目指しています。

## モニタリング計画
サイト内の生態系（供給サービス、文化的サービス、調整サービス）の維持・向上を確認するため、以下のモニタリングを実施します。

*   **対象分類群:** 昆虫類、植物、鳥類
*   **調査エリア:** ① 園庭ゾーン、② 農園ゾーン
*   **調査方法:** 直接観察または採取
*   **実施頻度・時期:** 原則として2年に1度。実施年の 2月、5月、8月、11月に実施。
*   **実施体制:** 愛管株式会社が活動計画を作成。同社社員、農園スタッフ、保育園職員が園児と一緒に観察を実施。
