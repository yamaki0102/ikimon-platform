## 設計原則

### 経営者の視点（ROI・リスク低減）
> 「これを入れると何が得られるか」「入れないと何を失うか」が3秒で分かること

### マーケターの視点（ファネル設計）
> 認知→興味→検討→試用→契約の各段階に最適化されたページ遷移

### 学者の視点（科学的信頼性）
> 手法の透明性がなければ企業は採用しない。「どう計算しているか」を隠さない

### 🌿 ウェルビーイングの視点（v2.1追加 / v2.2 学術根拠付与）

> [!IMPORTANT]
> **自然観察は「義務」ではなく「喜び」。** 観察会の本質は市民と社員の交流であり、それが幸福度と地域への愛着を自然に育む。この「副次効果」こそが企業価値の真のドライバーになる。

#### 3つの追加原則と学術的根拠

| 原則 | 意味 | サイト上の表現 |
|:---|:---|:---|
| **観察会 = 企業価値エンジン** | 生き物観察 → 市民交流 → 幸福度↑ → 地域愛着↑ → ブランド信頼↑ | 「TNFD対応」の前に「社員も地域もハッピーになる」を見せる |
| **教えない、気づかせる** | セミナー的な上から目線を排除。ドキュメントは「読まされる」のではなく「自然に読みたくなる」設計 | 専門用語は文脈で自然に理解できるよう配置。注釈やコラムで補足 |
| **環境担当者の味方** | 「俺、いい仕事してるじゃん」と実感できる。憧れではなく、**社内で正当に評価される武器**を渡す | レポートが上司への報告書にそのまま使える品質。数字と図表で成果を可視化 |

**役割**: ファネル最重要ページ。全B2Bトラフィックのエントリーポイント  
**CVゴール**: 「無料で始める」ボタンクリック or デモ閲覧

#### セクション構成（スクロール順序）

| # | セクション | 内容 | 心理的効果 |
|:--|:---|:---|:---|
| 1 | **ヒーロー** | 「観察するだけで、TNFDレポートが完成する」＋ CTA「無料で始める」 | 価値提案1文で完結。3秒ルール |

| 4 | **仕組み（How It Works）** | 3ステップ図解：①サイト登録 → ②社員・住民が**一緒に**観察 → ③自動レポート生成 | 「シンプルさ」を視覚で証明 |

#### コピーライティング指針
- **一人称は使わない** — "ikimonは〜"ではなく"あなたのサイトが〜になる"
- **数字を先に** — "90%のコスト削減"を文頭に
- **英語セクション併設** — 全セクションに英語版トグル。グローバル企業対応
- **モバイルファースト** — 意思決定者はスマホで初見する前提
- **教えない、気づかせる** — 専門用語を避けるのではなく、文脈で自然にわかる配置。注釈は読みたい人だけが開くアコーディオン形式
- **人の温度を入れる** — 制度の説明より先に「観察会で笑顔が生まれる」写真・ストーリーを見せる

### 5. `/methodology` — 計算手法の公開

**役割**: 科学的信頼性のアンカー。企業の内部稟議を通すための武器  
**CVゴール**: 直接CVではないが、「ここまで透明なら信頼できる」→ 間接CV

#### セクション構成

| セクション | 内容 |
|:---|:---|
| **BISスコア概要** | 100点満点の5軸構成をビジュアル図解 |
| **Species Richness (30点)** | Shannon-Wiener Index `H' = -Σ(pi × ln(pi))` の解説、正規化方法、参考論文リンク |
| **Data Confidence (20点)** | Research Grade比率 × 観察密度の定義、品質管理プロセス |
| **Conservation Value (25点)** | IUCNカテゴリ加重表、国際基準との整合性 |
| **Taxonomic Coverage (15点)** | 分類群カバレッジの算出方法、地域期待値の設定根拠 |
| **Monitoring Effort (10点)** | 季節カバレッジ、継続性の評価基準 |
| **限界と免責** | 市民参加型モニタリングの限界（検出バイアス、同定精度）を正直に記載 |
| **参考文献** | 引用論文・フレームワーク文書の一覧（学術的引用形式） |
| **更新履歴** | メソドロジーの変更ログ |

#### 設計上のポイント
- **LaTeX表記でなく平易な日本語で説明**。ただし数式は正確に記載
- **「なぜこの指標を選んだか」の理由**を各セクションに明記
- **英語版を同時公開** — グローバル企業のサステナビリティチームが読む前提
- **PDF版のダウンロード**提供 — 社内回覧用
