# SZ-UC-04 焼津港で見慣れない魚を共有する

## 1. 概要

- **目的**: 「詳しく分からないが共有したい」利用者が、海の観察を気軽に投稿し、反応を待てるかを検証する
- **主対象**: あいまい投稿、海洋文脈、コミュニティ期待値、海辺地点の自然さ
- **優先度**: 中高
- **想定所要時間**: 20 分

## 2. ユーザー像

- 焼津市の釣り好き、50 代
- 日常的に港へ行く
- 魚種には多少詳しいが、珍しいものは分からない
- 投稿目的は「珍しいかどうか知りたい」

## 3. シナリオ

9 月早朝、焼津港で見慣れない小型魚を見つけた。  
種類は分からないが、まずは写真を共有して詳しい人の意見を待ちたい。

## 4. 前提条件

- 写真 1〜2 枚を用意する
- 場所は港湾部として設定する
- 種名は `不明` もしくはざっくりした表現で進める

## 5. 必要な入力データ

- 仮ラベル: `見慣れない小型魚`
- メモ例: `岸壁近くの浅場。群れではなく単独でいた。`
- 日時: 9 月の早朝 5:30 頃
- 位置: 焼津港周辺

## 6. 検証タスク

- タスク A: 正体不明の魚を投稿する
- タスク B: 「詳しい人に見てもらいたい」期待を持てるか確認する
- タスク C: 海洋生物でも場違い感がないか確認する

## 7. 操作手順

1. 投稿画面を開く
2. 写真を追加する
3. 種名欄に曖昧な説明で進められるか確認する
4. 港の場所を設定する
5. 短い状況メモを入力する
6. 投稿する
7. 投稿後の詳細や一覧で見え方を確認する
8. コメントや同定支援の導線があるか確認する

## 8. 観察ポイント

### あいまいさの許容

- 種不明でも投稿の価値が損なわれないか
- 「こんな雑な情報では出せない」と感じさせないか

### 海洋文脈

- 魚やクラゲなど海の生き物でも自然に扱えるか
- 地点や分類の UI が陸上寄りに偏りすぎていないか

### 反応期待

- 他者が教えてくれそうな設計に見えるか
- 投稿後に孤立感がないか

## 9. 合格条件

- 種名不明でも 5 分以内に投稿できる
- 海辺の観察でも UI が自然に機能する
- 投稿後、「詳しい人が見てくれそう」と感じられる
- 地点の見え方が港湾利用者にも自然

## 10. 失敗シグナル

- 海の観察が例外扱いに見える
- 魚種が不明だと進めなくなる
- 投稿後に同定支援の期待を持てない
- 港の場所指定が扱いづらい

## 11. 深掘り質問

- 海の観察を歓迎している印象はあったか
- 「とりあえず上げてみる」が成立したか
- 投稿後に次の行動が思い浮かんだか
- コミュニティに見てもらえる感じはあったか
