# ikimon.life — Product Requirements Document (PRD)

> 最終更新: 2026-03-04（全ページ棚卸し済み）
> 現フェーズ: Phase 15B（Gamification & Personalization）

## 1. プロダクト概要

**ikimon.life** は市民参加型の生物多様性プラットフォーム。
30by30目標・TNFD LEAPフレームワークに対応し、市民科学（Citizen Science）と企業の生物多様性レポーティングを橋渡しする。

### ミッション
- 誰でも生きものの観察記録を投稿・共有できる
- 専門家による種同定（ID）で科学的品質を担保
- 企業の30by30/TNFD対応をデータドリブンで支援

### ターゲットユーザー
| セグメント | ペルソナ | 主要ニーズ |
|-----------|---------|-----------|
| 市民（toC） | 自然観察初心者〜中級者 | 記録・学習・コミュニティ |
| 研究者（toR） | 生態学・分類学の専門家 | データ品質・引用・DwC出力 |
| 企業（toB） | CSR/ESG担当・環境コンサル | 30by30レポート・TNFD LEAP・BIS |

## 3. モジュール構成

### 3.1 コア（観察 & フィード）
- 観察投稿（写真 + GPS + EXIF自動抽出）
- ホームフィード（最新・未同定・フォロー中）
- デイリークエスト
- オフラインキュー（PWA）

### 3.2 種同定（Identification）
- IDワークベンチ: プロ向け高速同定（旧IDセンター・未同定リストを統合）
- IDウィザード: 34検索テーブルによるガイド付き同定
- 種比較ビュー: 2種をOmoikane＋観察データで並列比較
- 信頼度レベル（menot / maybe / sure / literature）
- AI同定サジェスト（Omoikane連携）

### 3.3 リファレンス
- いきもの図鑑（13冊以上のフィールドガイドをデジタル化）
- 種ページ（741引用、分布図、レッドリスト状態）
- 参考文献レイヤー（1,248件の学術論文）

### 3.4 フィールド & ウェルネス
- フィールドリサーチ（GPSトラッキング）
- 歩数計・運動時間・気分トラッキング
- マイフィールド（活動履歴）

### 3.5 コミュニティ
- ユーザープロフィール & ライフリスト
- ゲーミフィケーション（ランキングページは削除済、バッジ・スコアで代替）
- イベント管理（RSVP）
- フォロー / いいね

### 3.6 B2B（企業向け）
- サイトダッシュボード（GeoJSON境界 + 種集計）+ 6ビューパーシャル（views/）
- 自治体 / エンタープライズ本社ダッシュボード（📦 アーカイブ、noindex）
- corporate_dashboard.php → showcase.php にリダイレクト統合
- BIS（Biodiversity Impact Score）
- TNFD LEAPレポート生成
- CSRショーケース / 埋め込みウィジェット
- DwC / GBIF エクスポート
- for-business/ サブサイト（LP / 申込 / 料金 / デモ / サンプルレポート）
- 料金: Community（無料） / Business（¥298k/年）

### 3.7 AI / Omoikane
- 10万種ナレッジグラフ生成・監視コンソール
- 種ナレッジ検索・ステータス API（ルートレベル配置）

### 3.8 管理 & モデレーション
- 管理画面（admin/）: ユーザー管理 / 観察管理 / モデレーション / Speed-ID検証 / 論文蒸留レビュー
- Freetextレビューキュー / グラント用レポート生成

### 3.9 ガイド記事（guide/）
- 8本のSEO・教育コンテンツ（ネイチャーポジティブ / 脳科学 / 認知症予防 / 企業事例 等）

### 3.10 情報ページ
- About / Pricing / Terms / Privacy / FAQ / Updates / Team / Guidelines
- XMLサイトマップ / オフラインページ / エラーページ（403, 404）

## 4. 非機能要件

- **PWA**: Service Worker, オフライン対応
- **レスポンシブ**: モバイルファースト、ボトムナビゲーション
- **パフォーマンス**: 画像WebP圧縮、CDN活用
- **プライバシー**: 希少種の位置情報マスキング
- **データ標準**: Darwin Core (DwC) 準拠エクスポート
- **多言語**: 日本語メイン（Lang クラスで管理）
