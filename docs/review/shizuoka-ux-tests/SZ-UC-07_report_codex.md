## 5. ケース観点ごとの評価

### 良かった点

- 位置・時刻・メモを保存する基礎は揃っている
- `explore.php` に外来種フィルタがある
- 詳細ページは記録単体を見るには十分

### 問題点

- 同地点比較の導線が弱い
- 一覧が単発観察前提に寄る
- 継続調査向けの緊張感ある整理UIが足りない

### 観察ポイント別メモ

- 精度: 投稿項目としては十分
- 再訪性: 目的に対して不足
- 研究補助性: 一歩足りない
