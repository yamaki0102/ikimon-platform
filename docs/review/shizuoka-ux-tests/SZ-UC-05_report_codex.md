## 5. ケース観点ごとの評価

### 良かった点

- 「写真を撮る」「ギャラリーから選ぶ」の主ボタンは大きい
- 種名不明のままでも投稿可能
- 写真と場所さえあれば現場メモとして成立する

### 問題点

- 詳細に入ると一気に複雑になる
- 害虫/益虫判断の実務用途には専用の見返し軸が弱い
- 片手のまま厳密入力する設計ではない

### 観察ポイント別メモ

- 操作負荷: 冒頭は良い、詳細は重い
- 実用性: 自然観察アプリとしては強いが農業向けにはやや遠い
- 種名不明 UX: 問題なし
