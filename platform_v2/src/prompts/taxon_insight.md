# Taxon Insight Prompt

`/api/v1/taxa/:name/insight` 等で使う。対象の生きもの（学名 / 和名）について、観察者が読んで「面白い」「もっと知りたい」となる豆知識を生成する。

## 入力
- 学名: ${scientificName}
- 和名: ${vernacularName}
- 観察地点: 緯度 ${lat} / 経度 ${lng}
- 季節: ${season}
- 観察者プロファイル: ${profileDigestSummary}

## プロファイル活用方針
`観察者プロファイル` が空でない場合のみ、`ecology_note` の最後に **1 文だけ** 観察者の関心領域に触れてよい (例: 「キミがよく見る湿地の植物群と同じ仲間」)。断定や個人特定情報は禁止、空のときは加味しない。

## 出力ルール

**UI にそのまま出る**ので、学術用語の羅列でなく、散歩中の市民観察者が「なるほど」と一息つけるトーンで書く。

- **etymology**（学名の由来）80字以内。属名の意味・種小名の意味を 1〜2 個解説。
- **ecology_note**（生態のツボ）100字以内。食性・生活史・面白い行動。「最新の研究では…」等の権威付けは避けて素朴に。
- **look_alike_note**（間違いやすい近縁種）80字以内。その生きものと混同しがちな種と、見分ける 1 ポイント。
- **rarity_note**（地域での出会いやすさ）60字以内。日本全域・季節でどこでどれくらい見られるか。絶滅危惧なら注意喚起。

## トーン

- 「〜だよ」「〜らしい」「〜と言われる」等、柔らかい語尾
- 学名ラテン語の読み解きは「ギリシャ語で『〇〇』の意味」のように親しみやすく
- 観察者の嬉しさに寄り添う視点（「見つけた人は割と運がいい」等、適度にポジティブ）
- 権威的な記述（「文献によると」「研究では」）は避ける

## 出力 JSON

```json
{
  "etymology": "属名 Aphis はギリシャ語で…",
  "ecology_note": "新芽の汁を吸って暮らす…",
  "look_alike_note": "アブラムシ類は全国に数百種。羽の有無で見分けやすい。",
  "rarity_note": "住宅地でも春〜秋によく出会える。"
}
```

JSON のみ。情報が乏しくて書けない場合は空文字で返す。
