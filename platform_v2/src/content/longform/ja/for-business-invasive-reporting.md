# 外来種候補の自動情報提供を、許可制で受け取る

ikimon.life は、市民や企業の投稿をもとに、外来種候補の情報を自治体・機関へ自動で届けられる仕組みを持ちます。ただし、送信は許可制です。公式サイトにメールアドレスが載っていても、受信許可がない窓口には自動送信しません。

このページは、自治体、県、国の機関、研究ネットワーク、海外の州政府・地方自治体が、どの条件で情報を受け取り、どの情報が共有され、どう止められるかを確認するための説明です。

## 自動で届く条件

次の条件がすべて揃った場合だけ、メール配信キューに入ります。

1. ユーザー投稿後のAI判定で、外来種 hard-gate を通っている。
2. 観察地点が、登録済みの管轄地域に入っている。
3. 種候補、上位分類、環境省カテゴリ、または地域ルールが、貴機関の募集内容に合っている。
4. 貴機関の連絡先が `approved` で登録されている。
5. 配信方式が `email` で、受信者レコードが有効になっている。

どれか1つでも欠ける場合、自動送信は行いません。内部監査では、候補化、送信抑止、pending、sent、failed を分けて残します。

## 共有される情報

共有対象は、外来種対応に必要な最小限の観察情報です。

- AI候補であること、確定同定ではないこと
- 観察日時
- 詳細位置、位置精度、場所メモ
- 写真URL
- 種候補、和名候補、学名候補、上位分類
- 環境省カテゴリ候補
- 個体数、観察メモ
- 窓口ごとに求められる情報
- 観察者に表示した注意事項

詳細位置は公開ページには出ません。公開ページでは原則として位置をぼかし、受信許可済みの機関に対する確認依頼だけに詳細位置を含めます。

## AI判定の限界

AI判定は、初動の候補化に使います。確定同定ではありません。

- 写真の角度、幼体・成体、花や実の有無で誤判定があります。
- 似た種類、園芸品種、在来近縁種との混同があります。
- 地域の指定状況や年度事業の対象は変わることがあります。
- 駆除、捕獲、処分、通報優先度の最終判断は、自治体・機関側の確認が必要です。

メール本文にも「AI候補であり、確定同定ではありません」と明記します。

## 受信開始・停止

受信を始める場合は、対象地域、対象種、分類、求める情報、注意事項、受信方法を確認します。

設定する項目は次の通りです。

| 項目 | 内容 |
|---|---|
| 管轄 | 国、都道府県、州、市町村、タイムゾーン、対応言語 |
| 窓口 | 機関名、部署名、メール、フォームURL、API、電話 |
| 許可状態 | approved、pending、not_requested、denied、external_only、revoked |
| ルール | 対象種、カテゴリ、緊急度、求める情報、注意事項、期間 |
| 監査 | candidate、suppressed_no_permission、pending_delivery、sent、failed |

停止したい場合は、連絡先を `revoked` または `denied` に変更します。停止後は新規の自動送信を行いません。公式フォームだけを案内したい場合は `external_only` にします。

## MECEな分類

| 分類 | 用途 | 例 |
|---|---|---|
| `emergency_biosecurity` | すぐ確認したい危険生物・侵入初期リスク | ヒアリ、アカカミアリ、カミツキガメ、exotic pest / disease |
| `sighting_report` | 分布把握、写真・場所・日時の情報提供 | オオキンケイギク、アライグマ、外来植物 |
| `management_consultation` | 防除、農地・水路・公園・施設管理 | 水草、つる植物、群落拡大 |
| `illegal_activity` | 違法放流、違法飼養、移動・譲渡 | 生体の放流や販売の疑い |
| `time_limited_program` | 年度調査、駆除イベント、期間限定募集 | 市のいきもの調査、外来植物駆除作戦 |

この分類により、緊急通報と年度イベントを混ぜず、海外の partner API や外部フォームにも拡張できます。

## 国際連携の前提

データモデルは日本語だけに寄せていません。`country_code`、`admin_area_1`、`admin_area_2`、`municipality`、`languages`、`timezone` を分け、メール、外部フォーム、電話、パートナーAPIを別の接続方式として扱います。

EU / EASIN、米国 / EDDMapS、カナダ・オンタリオ州、ニュージーランド Biosecurity、オーストラリアやニュージーランドの地方自治体のように、外来種通報の入口は国ごとに違います。v1では、公式フォームやアプリへ勝手に代理送信せず、許可済みメール連携だけを自動送信します。API連携は `partner_api` として、相手先仕様と同意が確認できた地域から有効化します。

## 確認したい場合

受信設定、対象種、文面、停止条件、言語、API連携の相談はお問い合わせください。既存の公式窓口だけを案内したい場合も、自動送信なしの `external_only` として登録できます。
