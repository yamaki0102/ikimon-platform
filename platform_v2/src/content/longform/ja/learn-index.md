# 読みもの

読みたいものを、近い入口から選んでください。読み物は、基本的に1テーマ1ページです。用語は用語ページへ、考え方は解説ページへ分けています。

## はじめて使う

[名前が分からないとき](/learn/identification-basics)
名前を決めきれないままでも、あとで確かめやすい記録を残す方法。

[観察の流れ](/learn/field-loop)
見つける、記録する、あとで読む、また歩く流れ。

[生物多様性の基礎](/learn/biodiversity)
生物多様性を、暮らしの中の観察から読むページ。

[市民科学と観察](/learn/citizen-science)
市民の観察が、科学や地域の見守りに近づく条件。

[BioMonWeekで観察する](/learn/biomonweek)

[記録・データ・信頼性について](/learn/methodology)

[政策・企業活動と自然](/learn/policy-and-business)
