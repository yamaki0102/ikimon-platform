# Darwin Core

Darwin Core は、生物多様性データを交換しやすくするための標準語彙です。

分類名、日付、場所、観察者、証拠、識別子など、観察や標本を外部と共有するときに必要な項目をそろえるために使われます。

## ikimon.lifeでの扱い

研究利用や外部共有では、ただ文章で説明するだけでなく、項目の意味をそろえる必要があります。Darwin Core は、そのための共通語彙として扱います。
