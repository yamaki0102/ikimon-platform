# 外来種候補を見つけたとき、どこへ何を伝えるか

外来種の情報提供は、全部を同じ窓口へ送ればよいものではありません。危険生物の緊急通報、写真と場所の発見情報、防除相談、違法行為の通報、年度調査や駆除イベントでは、求められる情報と注意点が違います。

ikimon.life では、AIが外来種候補を出したあと、地域のルールと受信許可が揃った場合だけ、自治体・機関へ観察情報を自動で共有します。受信許可がない窓口には自動送信しません。その場合は、ユーザーに公式窓口を案内し、内部の監査ログには「送信抑止」として残します。

## まず守ること

- 素手で触らない。危険生物、毒を持つ生物、とげのある植物は距離を取る。
- 生きたまま運ばない。特定外来生物は、保管・運搬・放出が規制対象になることがあります。
- 駆除判断をしない。AI候補は確定同定ではありません。駆除、移動、処分は自治体・管理者の確認後に行います。
- 写真、発見場所、日時、個体数、広がりを残す。安全に撮れる範囲で十分です。
- 水草やつる植物は破片を散らさない。切れ端や土で広がる種類があります。

## MECE分類

| 分類 | 代表例 | 求められる情報 | 注意 |
|---|---|---|---|
| 緊急通報 | ヒアリ、アカカミアリ、ゴケグモ類、カミツキガメ、海外では exotic pest / disease | 写真、詳細位置、日時、個体数、大きさ、安全上の状況 | 触らない、近づかない、捕まえない |
| 写真・発見情報提供 | オオキンケイギク、アライグマ、外来植物全般 | 写真、場所、日時、個体数または範囲 | 同定が弱い場合は候補として伝える |
| 防除協力・相談 | 水路、農地、公園、企業敷地の外来植物 | 場所、広がり、管理者、作業予定、写真 | 切れ端、種子、土を移動させない |
| 違法行為 | 違法放流、違法飼養、移動・譲渡 | いつ、どこで、何が、誰が分かる範囲で | 個人で追跡・接触しない |
| 年度事業 | 市のいきもの調査、駆除イベント、期間限定募集 | 事業名、対象種、対象期間、写真、場所 | 募集期間と対象地域を確認する |

## 自動で送られる条件

次のすべてを満たした場合だけ、自動で自治体・機関に送ります。

1. 投稿後のAI判定で、外来種 hard-gate を通っている。
2. 観察地点が、登録済みの地域ルールに入っている。
3. 対象種またはカテゴリが、窓口の募集内容に合っている。
4. 送信先が `approved` で、メール連携先として受信許可済み。

許可がない場合は送信しません。画面では公式窓口を案内し、内部では `suppressed_no_permission` として監査します。

## 自動共有される情報

- AI候補であること、確定同定ではないこと
- 観察日時
- 詳細位置、位置精度、場所メモ
- 写真URL
- 種候補、上位分類、環境省カテゴリ候補
- 個体数、観察メモ
- その窓口が求める情報
- 観察者へ表示している注意事項

詳細位置は、自治体・機関への確認依頼では使いますが、公開ページでは原則ぼかして表示します。

## 国際対応

海外の外来種通報は、国や地域によって窓口の形が違います。ikimon.life のデータモデルは、国コード、州・県、市町村、対応言語、タイムゾーン、メール、外部フォーム、パートナーAPIを分けて持ちます。

- EU / EASIN は、Web / App から市民報告を受け、位置、1から3枚の写真、専門家確認、加盟国当局との接続を前提にしています。参考: [EASIN Report Species](https://easin.jrc.ec.europa.eu/ias), [EASIN Member State Authorities](https://easin.jrc.ec.europa.eu/easin/MSCAuthorities)
- 米国 / EDDMapS は、全国的な専門確認ネットワークとモバイル報告を使います。参考: [USDA NISIC Reporting](https://www.invasivespeciesinfo.gov/subject/reporting), [EDDMapS](https://www.eddmaps.org/?lang=en)
- カナダ・オンタリオ州は、EDDMapS、アプリ、違法行為の電話通報を分けています。参考: [Ontario invasive species](https://www.ontario.ca/page/invasive-species-ontario)
- ニュージーランドは exotic pest / disease を Biosecurity New Zealand へ報告し、緊急性が高いものは電話を優先します。参考: [MPI report a pest or disease](https://www.mpi.govt.nz/biosecurity/how-to-find-report-and-prevent-pests-and-diseases/report-a-pest-or-disease)
- オーストラリアやニュージーランドの地方自治体は、目撃情報を地図化、バイオセキュリティ対応、教育に使います。参考: [Environment Canterbury](https://www.ecan.govt.nz/your-region/your-environment/biodiversity-and-biosecurity/report-invasive-plant-or-animal-sightings), [Australia DCCEEW invasive species](https://www.dcceew.gov.au/environment/invasive-species)

v1では、海外の外部フォームや公式アプリへ無理に自動送信しません。`external_form` や `partner_api` として接続方式を分け、相手先の許可と仕様が確認できた地域から有効化します。
