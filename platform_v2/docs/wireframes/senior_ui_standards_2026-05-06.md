# Senior UI Standards 2026-05-06

目的: 年配/熟練モニター、低視力、屋外利用、片手操作を前提に、ikimon.life のトップ/観察一覧で守るUI基準を定義する。

対象:

- トップページ
- 観察投稿一覧
- 同定待ちカード
- 地域マップ棚

## 1. 基本方針

高齢者向けUIは、単に文字を大きくするだけでは足りない。

- 何を押せばよいかを減らす
- 状態を色だけで伝えない
- 写真と短いラベルで判断させる
- 説明はサブページへ逃がす
- 操作地図や細かいフィルタをトップに詰め込まない

## 2. 文字サイズ

| 用途 | 基準 |
|---|---|
| 本文 | 17px / line-height 1.7 |
| 最小テキスト | 14px未満禁止。補足・日付・場所でも14pxを基本 |
| カード見出し | 17-20px |
| セクション見出し | 20-24px |
| ページ見出し | 28-34px |
| 状態ラベル | 13-14px、太字 |

実装ルール:

- ビューポート幅連動で文字サイズを連続変化させない。
- モバイル/デスクトップはブレークポイントで固定サイズを切り替える。
- 日本語の字間を詰めない。letter-spacing は原則 0。

## 3. タップ領域

| 要素 | 最小 |
|---|---|
| 主要CTA | 高さ56px以上 |
| トップ4大入口 | 高さ76px以上 |
| アイコンボタン | 52px以上 |
| フィルタ/タブ | 高さ48px以上。主要タブは56px推奨 |
| 観察カード | 縦100px以上、写真付きは118px以上 |
| チェックボックス/トグル | ラベル込みで56px以上 |

実装ルール:

- 44pxアイコン単体をモバイル主要操作に使わない。
- アイコンだけにせず、未習熟ユーザーには文字ラベルを併記する。
- ボタン間隔は8px以上。

## 4. カード基準

トップと観察一覧のカードは、情報を詰め込まない。

| 要素 | 基準 |
|---|---|
| 写真 | 必須に近い主役。写真なしは明確なプレースホルダー |
| 見出し | 1行基本、最大2行 |
| 補足 | 場所丸め + 日付 |
| 状態 | 1つだけ強調。複数状態は詳細画面へ |
| 説明文 | トップカードでは原則なし |

## 5. 状態ラベル

内部ステータスをそのまま出さない。

| 内部/概念 | 表示 |
|---|---|
| AI candidate | AI候補 |
| needs identification | 同定待ち |
| community review | 確認中 |
| trusted/expert reviewed | 確認済み |
| needs_more_evidence | 追加写真が必要 |
| review_ready | 確認しやすい記録 |
| monitoring_ready | 調査に使いやすい記録 |

禁止:

- AI候補を `確定` と表示する。
- 色だけで状態を伝える。
- `review_ready` など内部英語を表示する。

## 6. トップページ固有基準

- 第1ビューの主行動は `観察する`, `同定する`, `地図`, `マイページ` の4つ。
- ヘッダーのブランド表示は、現行ロゴマーク + 提供された ikimon 文字ロゴを組み合わせる。
- `再訪問` は主CTAにしない。
- 地域マップ棚で、近くの変化・前回記録・季節差分を見せる。
- 読み物は1棚まで。
- 制度説明、研究向け説明、企業向け説明はトップに置かない。
- ページ最下部の既存フッターは維持してよい。
- スマホ下部の固定導線は、トップ4大入口と競合しないよう `すぐ記録する` に役割を限定する。

## 7. 観察一覧固有基準

採用案: **Video Grid Style**

- 写真サムネイルを大きくする。
- フィルタは `すべて`, `同定待ち`, `近く`, `写真あり`, `地域マップ` 程度に絞る。
- スマホでは2列か1列。文字が詰まる場合は1列を優先。
- Map/List/Gridの多機能切替は後段でよい。

## 8. 検証基準

実装後に見る項目。

- 360px幅で横スクロールが出ない。
- 主要CTAが親指で押せる。
- 文字サイズ14px未満がない。
- カード内の長い種名がUIを壊さない。
- 状態ラベルが色なしでも意味を持つ。
- 地図がトップ表示を重くしない。
- キーボードフォーカスが見える。
- `npm --prefix platform_v2 run typecheck` が通る。
