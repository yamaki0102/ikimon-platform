| ⬜ 未着手 | まだ作業を開始していない |
| 🔄 進行中 | 一部ページが処理済み |
| ✅ 完了 | 全ページ処理済み（検証も完了） |
| ⏸️ 保留 | 仕様検討中、または優先度低 |
