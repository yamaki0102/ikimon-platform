# The Road to the Strongest Bio-Navigator: Mass Ingestion Strategy

You asked: *"How do we handle 100+ books and papers to create the Strongest Bio-Navigator?"*
The answer lies in treating ingestion not as "Digitization" but as **"AI Training"**.

## 1. The "Strongest" Definition
Standard AIs (ChatGPT, Gemini) know "General Biology".
**The ikimon Bio-Navigator** will know **"Contextual Reality"**.

*   **Standard AI**: "This looks like a Stag Beetle."
*   **ikimon Navigator**: "This is a *Prosopocoilus inclinatus*. Since you are in **Sagamigawa (Map Data)** in **August (Season Data)**, and it has **L-shaped antennae (Logic Data)**, identification is 99% certain. Historically, children in this region called it 'Suigyū' (**Cultural Data**)."

## 2. Ingestion Strategy for Scale (The 4-Tier Pipeline)
To feed this Navigator, we must ingest data in layers.

### Tier 1: The Core Logic (Handbooks) ~20 Books
*   **Goal**: Diagnostic capability.
*   **Strategy**: **Agent-Driven Logic Mining**. Extract "Decision Trees" (e.g., "If mandible is curved...").
*   **Output**: The "Brain" of the Navigator.

### Tier 2: The Spatio-Temporal Grid (Regional Guides) ~30 Books
*   **Goal**: Precision Filtering.
*   **Strategy**: **Hybrid Map Mining**. Extract "Where" and "When".
*   **Output**: The "GPS" of the Navigator. (Eliminates candidates that don't live in the user's area).

### Tier 3: The Deep Mechanism (Papers/Magazines) ~50+ Items
*   **Goal**: "Why" & "How".
*   **Strategy**:
    *   **Magazines (Insectarium)**: **Methodology Mining** (Rearing, Collecting).
    *   **Papers**: **Automated PDF Mining** (Standardized Facts).
*   **Output**: The "Teacher" personality. (Explains mechanisms like photoperiodism).

### Tier 4: The Soul (Cultural/Historical Books) ~10+ Books
*   **Goal**: Emotional connection.
*   **Strategy**: **Entity Mining**.
*   **Output**: The "Storyteller". (Connects biology to human history).

## 3. Future Development: The "Bio-Graph"
We are not just making JSON files. We are building a **Knowledge Graph (Bio-Graph)**.
*   Everything is connected via `modern_taxon_id`.
*   **Paper A** (2005) connects to **Book B** (1995) connects to **Map C** (2010).
*   **Temporal Honesty**: The Navigator knows *when* each fact was true, allowing it to explain changes (e.g., "This insect used to be here in 1990, but papers show it vanished by 2005").

## 4. Execution Protocol
1.  **Agent Mode (Current)**: To crack the complex books (Tier 1, 2, 4) and establish the schema.
2.  **Batch Automation (Future)**: Once the schema is rock-solid, we use scripts for simple Tier 3 (Papers).
3.  **Vectorization**: All V3 JSONs are instantly vectorized for RAG (Retrieval Augmented Generation).

**Conclusion**: Proceeding with this 100-book plan is the **only way** to build an AI that surpasses generic models. We are building the *specialized training dataset* that no one else has.
