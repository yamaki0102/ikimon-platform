「精度より量」ではなく「精度は最高で保存、量も最大化、表示は権限で制御」が正解。

**Why:** ユーザーによって必要な粒度が違う。100年後の科学者は1kmグリッドで十分だが、自然共生サイト管理者や漁業者・山林所有者はピンポイントが必要。

**How to apply:**
- 記録時: GPS生データ（±5m）をそのまま Canonical Schema に保存
- 公開マップ: デフォルト100mグリッド（プライバシー＋希少種保護）
- サイト管理者: 管理区域内ならピンポイント閲覧可能
- 研究者/行政: アクセス申請で精密座標を提供
- PrivacyAccess レイヤーの coordinate_precision で制御（実装済み）
- GPS精度が低くてもデータを捨てない。coordinate_uncertainty_m に正直に記録するだけ
