# SEO & AIコンテンツ戦略設計書
## 種ページを検索上位に押し上げる仕組み

---

## 1. 目標

**「アオスジアゲハ 浜松」で検索したら ikimon が1位**

---

## 2. SEO基盤設計

### 2.1 URL構造

```
/species/papilio-xuthus          ← 学名ベースのURL（永続性）
/species/papilio-xuthus/hamamatsu ← 地域×種の組み合わせページ
```

### 2.2 構造化データ（Schema.org）

```json
{
  "@context": "https://schema.org",
  "@type": "Species",
  "name": "アオスジアゲハ",
  "scientificName": "Graphium sarpedon",
  "description": "...",
  "sameAs": ["https://www.gbif.org/species/1933046"]
}
```

---

## 3. AI活用によるコンテンツ自動生成

### 3.1 AIを使う領域（著作権リスクなし）

| コンテンツ | 生成方法 | 安全な理由 |
|-----------|---------|-----------|
| **季節カレンダー** | 観察データから自動集計 | ファクトベース |
| **観察数・分布マップ** | 観察データから自動生成 | ファクトベース |
| **「一緒に見られる種」** | 同一場所の観察から集計 | ファクトベース |
| **月別グラフ** | 観察データから自動生成 | ファクトベース |

### 3.2 外部ソースからの引用（著作権クリア）

| ソース | ライセンス | 引用方法 |
|--------|-----------|---------|
| **Wikipedia日本語版** | CC-BY-SA 3.0 | 引用元を明記 + 同じライセンスで公開 |
| **Wikispecies** | CC-BY-SA | 学名・分類情報の取得 |
| **GBIF Species Profiles** | CC-BY 4.0 | 引用元を明記 |

**表示例**:
```
アオスジアゲハは、翅に青緑色の帯模様を持つ美しいアゲハチョウです。
都市部の公園でもよく見られ、クスノキなどを食草とします。
— 出典: Wikipedia「アオスジアゲハ」(CC-BY-SA 3.0)
```

### 3.3 AIを使わない領域

| コンテンツ | 対応 |
|-----------|------|
| **オリジナル解説文** | ❌ AI不使用。Wikipedia引用 or 専門家手動執筆 |
| **見分け方ガイド** | ❌ AI不使用。専門家が手動で執筆 |

### 3.4 コンテンツ生成ワークフロー

```
[新種ページ作成時]
Step 1: GBIF/Wikipediaから基本情報取得
Step 2: AI（GPT-4o）で日本語要約
Step 3: 管理者確認（必須）
Step 4: 公開
```

### 3.3 AIが生成するコンテンツ例

```markdown
## アオスジアゲハ (Graphium sarpedon)

### 浜松市での観察状況
- **確認回数**: 127回（2024年）
- **よく見られる場所**: 佐鳴湖公園
- **ピークシーズン**: 5月〜9月

### 観察のコツ
> 「クスノキの周りをパトロールしていることが多いです」
```

## 4. 地域×種ページ（ロングテール戦略）

観察データが10件以上ある組み合わせで自動生成:

```
/species/graphium-sarpedon/hamamatsu
→ 「浜松市のアオスジアゲハ」専用ページ
```

コンテンツ: 地域マップ、月別グラフ、ベスト写真、一緒に見られる種
