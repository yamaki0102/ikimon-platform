# 利用規約・プライバシーポリシー設計
## ikimon サービス利用に関する法的文書

---

## 1. プライバシーポリシー（要点）

### 1.1 収集する情報

| カテゴリ | 情報 | 目的 |
|---------|------|------|
| **アカウント情報** | メール、氏名（任意）、SNS連携ID | 認証、通知 |
| **位置情報** | GPS座標（写真Exif or 手動入力） | 観察地点の記録 |
| **デバイス情報** | ブラウザ、OS、IPアドレス | セキュリティ、分析 |
| **行動情報** | 投稿履歴、同定履歴 | サービス改善 |

### 1.2 位置情報の取り扱い

```
[重要な開示事項]
- 希少種の位置は自動的に秘匿されます（10km/1kmグリッドにランダム化）
- 一般種の位置は公開されます
- 位置情報を非公開にしたい場合は手動で精度を変更できます
```

### 1.3 データの第三者提供

| 提供先 | 目的 | 同意 |
|--------|------|------|
| **GBIF** | 学術研究・保全活動 | CC-BY 4.0（デフォルト） |
| **自然共生サイト申請** | OECM/30by30 | 有料契約者のみ |
| **研究機関** | 学術論文 | CC-BY 4.0 |

> ユーザーは投稿時にライセンスを選択可能（CC-BY or CC-BY-NC）

### 1.4 データの保持期間

| データ | 保持期間 |
|--------|---------|
| 観察データ | 永続（科学的価値のため） |
| アカウント情報 | 退会後30日で削除 |
| 写真 | 永続（削除リクエスト可） |

### 1.5 ユーザーの権利

- **アクセス権**: 自分のデータをダウンロード可能
- **訂正権**: 誤った情報を修正可能
- **削除権**: 特定の観察を削除可能（Research Gradeは要審査）
- **反対権**: データのGBIF提供を拒否可能

## 2. 利用規約（要点）

### 2.1 サービス利用条件

- 13歳以上が利用可能
- 正確な観察情報を投稿すること
- 他者の権利を侵害しないこと

### 2.2 禁止事項

| 禁止行為 | 対応 |
|---------|------|
| 希少種の位置を暴露する行為 | 即時BAN |
| 虚偽の観察データ投稿 | 警告 → BAN |
| 他ユーザーへの嫌がらせ | 警告 → BAN |
| 商用利用（規約外） | 警告 → BAN |
| 自動投稿ボット | 即時BAN |

### 2.3 コンテンツの権利

```
[写真の権利]
- 著作権: ユーザーに帰属
- ライセンス: ikimonに非独占的使用権を付与
- 第三者提供: ユーザーが選択したCC-BYライセンスで提供
```

### 2.4 免責事項

- 同定の正確性は保証しない
- サービス停止による損害は責任を負わない
- ユーザー間のトラブルは当事者間で解決

### 2.5 有料プラン

- 支払い: **銀行振込のみ**（初期）
- 契約期間: 年間契約
- 返金: 原則不可（サービス不具合時は検討）

## 3. 法人アカウント管理

### 3.1 アカウント構造

```
[法人アカウント]
├─ 組織 (Organization)
│   ├─ 管理者1（オーナー）
│   ├─ 管理者2
│   └─ 管理者3
└─ サイト1, サイト2, ...
```

### 3.2 権限レベル

| 権限 | オーナー | 管理者 | 閲覧者 |
|------|---------|--------|--------|
| サイト登録依頼 | ✅ | ✅ | ❌ |
| 担当者情報編集 | ✅ | ✅ | ❌ |
| 管理者追加/削除 | ✅ | ❌ | ❌ |
| 支払い情報変更 | ✅ | ❌ | ❌ |
| レポート閲覧 | ✅ | ✅ | ✅ |
| オーナー変更 | ✅ | ❌ | ❌ |

### 3.3 担当者退職時の対応

```
[オーナー退職時]
1. 既存オーナーが新オーナーを指名
2. 指名できない場合はIKIMON社に連絡
3. 本人確認後、新オーナーを設定

[管理者退職時]
1. オーナーが管理者を削除
2. 新管理者を追加
```

### 3.4 招待フロー

```
1. オーナーが管理者のメールアドレスを入力
2. 招待メール送信
3. 招待を受けた人がログイン/登録
4. 権限付与完了
```

## 4. 決済フロー（初期）

### 4.1 銀行振込のみ

```
[契約フロー]
1. サイト登録依頼フォーム送信
2. IKIMON社が請求書発行（PDF/メール）
3. 銀行振込で支払い
4. 入金確認後、サイト登録完了
5. 以降、年1回更新請求
```

### 4.2 請求情報

| 項目 | 内容 |
|------|------|
| 振込先 | IKIMON株式会社 指定口座 |
| 支払期限 | 請求書発行から30日以内 |
| 更新通知 | 契約終了60日前にメール |
