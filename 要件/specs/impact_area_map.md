# Impact (エリアマップ) 仕様書

## 概要

ikimonの「Impact」ページは、ユーザーの**活動エリア**を中心に、集団効力感と科学的貢献を可視化するシステムです。

---

## コアコンセプト

### 全人類 vs 空白エリア (Ingress風)
- 敵チームは存在しない。「空白（未観察エリア）」が共通の敵。
- ユーザーは観察データで「地図を埋める」協力プレイに参加。
- 30by30（2030年までに陸海30%保全）に直接貢献。

---

## 住所階層サポート

### 対応レベル

| レベル | 名称 | 例 | 対応 |
|-------|------|-----|------|
| L0 | 国 | 日本 | ✅ |
| L1 | 都道府県 | 静岡県 | ✅ |
| L2 | 市区町村 | 浜松市、森町、新城市 | ✅ |
| L3 | 区（政令市のみ） | 中区、浜北区 | ✅ |
| L4 | **町・大字・村** | 北寺島町、都田町 | ✅ |
| L5 | 丁目・番地 | 1丁目2番3号 | ❌ 粒度過剰 |

### データソース
- **国土数値情報「行政区域データ」**: 無料、GeoJSON
- **逆ジオコーディング**: 観察座標から動的に取得

## ファーストビューのパーソナライズ

### ホームエリア決定ロジック

```
user.home_area = mode(投稿位置の行政区域)
user.activity_spread = count(distinct 市区町村)

if 投稿なし:
    default_view = L1 (県)
elif activity_spread >= 3:
    default_view = L1 (県)
elif activity_spread == 1 and 全投稿が同一町内:
    default_view = L4 (町・大字)
else:
    default_view = L2 (市区町村)
```

### ユーザータイプ別デフォルト

| ユーザータイプ | デフォルト表示 | 判定条件 |
|--------------|--------------|---------|
| 新規ユーザー | 県レベル | 投稿なし |
| 広域活動者 | 県レベル | 3市区町村以上に分散 |
| 市域活動者 | 市区町村レベル | 1-2市区町村に集中 |
| 局所活動者 | 町・大字レベル | 投稿の80%が特定町内 |

## 自動生成ミッション

### ロジック
- 現在表示中のエリアで最もカバレッジが低いサブエリアを抽出
- テンプレート文で自動生成

### テンプレート
```
タイトル: "{サブエリア}の生き物マップを埋めよう"
説明: "カバレッジ率が{X}%で、このエリア内で最も空白が多い地域です。"
```

## 今月の貢献者 (Top 10%)

### 部門
- **観察部門**: 最も多く観察を投稿した人
- **同定部門**: 最も多く他の投稿を同定した人

### 表示ルール
- 順位ではなく「Top 10%」バッジのみ表示
- エリア単位で集計（選択中のエリア内での貢献）
- タップでプロフィールへ → いいね導線
