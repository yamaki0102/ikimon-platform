## 2. 外来種アラート機能

### 2.1 対象種

| カテゴリ | 例 | アラートレベル |
|---------|-----|-------------|
| **特定外来生物** | アライグマ、オオクチバス | 🔴 最重要 |
| **生態系被害防止外来種** | アメリカザリガニ | 🟡 重要 |
| **その他外来種** | セイヨウタンポポ | 🟢 情報提供 |

### 2.2 アラート表示

```
[投稿時]
⚠️ この種は「特定外来生物」に指定されています。

外来生物法により、飼育・運搬・放流は禁止されています。
詳細: https://www.env.go.jp/nature/intro/

[記録として非常に重要です！発見情報をありがとうございます]
```

### 2.3 データ活用

- 外来種分布マップの自動生成
- 自治体への情報提供（API）
- 駆除活動団体への通知（オプトイン）

### 2.4 マスタデータ

環境省の「生態系被害防止外来種リスト」をマスタ化:
- 約430種を登録
- GBIF Backboneと紐付け
- 週次で更新確認

## 3. ゲーミフィケーション設計

### 3.1 設計原則

> **競争より協力、外的報酬より内的報酬。**

| 避けるべき設計 | 採用する設計 |
|--------------|------------|
| ランキング上位争い | 地域での達成率 |
| ポイント交換商品 | 知識・スキルの可視化 |
| 他者との比較 | 過去の自分との比較 |

### 3.2 達成システム

| カテゴリ | 達成項目 | 条件 |
|---------|---------|------|
| **観察** | 初めての投稿 | 1件投稿 |
| | 100種達成 | ライフリスト100種 |
| | 地域コンプリート | 指定地域で50%の種を確認 |
| **同定** | 初めての同定 | 1件同定 |
| | 分類群博士 | 特定分類群で上位5% |
| | Research Grade貢献 | 10件達成に貢献 |
| **シーズン** | 春の訪れ | 春の指標種を確認 |
| | 夏の虫博士 | 夏の昆虫10種確認 |

### 3.3 バッジシステム

| バッジ | 条件 | 表示 |
|--------|------|------|
| 🔬 認定研究者 | IKIMON社審査通過 | プロフィールに常時表示 |
| 🦋 昆虫博士 | 昆虫で上位5% | プロフィール+同定時 |
| 🐦 鳥類博士 | 鳥類で上位5% | 同上 |
| 🌿 植物博士 | 維管束植物で上位5% | 同上 |
| 📸 写真の達人 | 高評価写真10枚 | プロフィール |
| 🏅 Research Grade達成者 | 100件貢献 | プロフィール |

### 3.4 ポイント設計（内部値）

> ポイントは**非公開**。ランキング表示には使わない。

| アクション | ポイント | 目的 |
|-----------|---------|------|
| 観察投稿 | +1 | 投稿促進 |
| 同定投票 | +1 | 同定促進 |
| Research Grade貢献 | +5 | 質の高い同定 |
| 正しい同定（後から確認） | +3 | 精度向上 |
| 誤同定（後から覆される） | -1 | 軽いペナルティ |

### 3.5 モチベーションの種類別設計

| ユーザータイプ | モチベーション | 対応機能 |
|--------------|-------------|---------|
| **収集家** | 「全部集めたい」 | ライフリスト、地域図鑑 |
| **専門家志向** | 「詳しくなりたい」 | 博士バッジ、同定精度表示 |
| **貢献者** | 「役に立ちたい」 | Research Grade、CSRショーケース |
| **社交家** | 「仲間がほしい」 | 地域コミュニティ、イベント |
| **探検家** | 「新しい発見」 | 初記録通知、レア種アラート |

### 3.6 避けるべきパターン

| パターン | 問題点 | ikimonの対応 |
|---------|--------|------------|
| **リーダーボード** | 上位以外のやる気低下 | 地域達成率のみ |
| **デイリーログイン報酬** | 義務感、形骸化 | 採用しない |
| **ガチャ・ランダム報酬** | 射幸心、依存 | 採用しない |
| **ポイント有効期限** | ストレス | 採用しない |
